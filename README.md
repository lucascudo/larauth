## Server Requirements
- PHP >= 7.1.3
- OpenSSL PHP Extension
- PDO PHP Extension
- Mbstring PHP Extension
- Tokenizer PHP Extension
- XML PHP Extension
- Ctype PHP Extension
- JSON PHP Extension

Directories within the ***storage*** and the ***bootstrap/cache*** directories should be writable by your web server

## Installing
```sh
composer install
```

## Running
```sh
php artisan serve
```
